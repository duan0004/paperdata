\section*{S1. Data Products and Density Convention}

The analysis uses public NANOGrav 15-year data products
(arXiv:2306.16213), the NANOGrav 15-year noise analysis
(arXiv:2306.16214), and the NANOGrav new-physics model-comparison paper
(arXiv:2306.16219).  Official NANOGrav/PTArcade model files were taken
from Zenodo DOI 10.5281/zenodo.8084351, and the official PTArcade
\texttt{ceffyl} density product from Zenodo DOI 10.5281/zenodo.10495907.

PTArcade model files return $h^2\Omega_\mathrm{GW}$.  The residual-density
conversion used for official templates is
\begin{equation}
\rho^2 =
\frac{(H_0/h)^2 h^2\Omega_\mathrm{GW}}
{8\pi^4 f^5 T_\mathrm{span}} .
\end{equation}
This conversion was checked against \texttt{ptarcade.models\_utils.omega2cross}
to relative precision below $5\times10^{-16}$.

\section*{S10. Evidence Cross-Check}

The four main non-baseline evidence rows were cross-checked independently of
dynesty by evaluating the official-density likelihood on Sobol points in the
prior hypercube.  For each model we computed both the direct estimator
$\log Z=\log E_\mathrm{prior}[L]$ and the power-posterior thermodynamic
integration estimator
\begin{equation}
\log Z=\int_0^1 \langle \log L\rangle_\beta\,d\beta .
\end{equation}
The run used 16384 Sobol points per scramble, five independent scrambles, and
81 $\beta$ values.
